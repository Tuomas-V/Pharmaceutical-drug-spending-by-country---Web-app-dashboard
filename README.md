# Pharmaceutical drug spending by country - Web app dashboard
Interactive web app dashboard in which you can choose to display data by selected countries.

<img src="/Screenshots/Output1.png">

The dashboard reads data from an CSV file and is hosted as HTML-based web app.</br>
User can freely select any number of countries from the dropdown menu to be displayed.
The graphs automatically color each country and follow the same selections.

<img src="Screenshots/Output2.png">

The graphs are made using Plotly Express and Dash in python, which allows easy source of
custom hover effects and automatic updates each time user changes selection of items.

<img src="Screenshots/Output3.png">

As it is an HTML-based web app, it allows CSS styles.

#### Data sources:
 - [Datahub.io](https://datahub.io/core/pharmaceutical-drug-spending)
